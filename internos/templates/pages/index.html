WHAT IS NEURO-DB?
Neuro-DB is a data warehouse/centralized online database tool to gather, organize, manage, map and analyze data from different sources of UNICEF programmes. The objective is to create sort of an inter-relation between these data that allow staff to access information in an easy and secured way and to generate customized reports based on data from different programmes/sources that can help UNICEF staff to make a better decision based on evidence base data/indicators.
WHAT IS NEURO-DB?
As the graph shows below, Neuro-DB will be equipped by an internal engine to fetch data from external databases as Input and provide a new indicator as Output.
  • Data/Indicators from different sources as an Input
  • Linking between these data in the middle
  • Coming up with new Indicators as Output
