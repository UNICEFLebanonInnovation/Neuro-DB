WHAT IS NEURO-DB?
Neuro-DB is a data warehouse/centralized online database tool to gather, organize, manage, map and analyze data from different sources of UNICEF platforms, databases, Information Management Systems and programmes. The objective is to create a one-stop shop center that brings together inter-relation among these databases and platforms in order to allow staff to access information in an seamless, user friendly and secured manner.
Neuro-DB also generates customized reports based on data from different sources that can help UNICEF staff to make better decisions based on evidence based real-time data/indicators.
WHAT IS NEURO-DB?
As the graph shows below, Neuro-DB will be equipped by an internal engine to fetch data from external databases as Input and provide a new indicator as Output.
  • Data/Indicators from different sources as an Input
  • Linking between these data in the middle
  • Coming up with new Indicators as Output
